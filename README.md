## TTime 是什么？

TTime 是一款跨平台的 AI 翻译软件，支持 Windows 和 macOS。核心思路很简单：**不管你在看什么，不管文字在哪，按个快捷键就能翻译**。

不用切应用，不用复制粘贴，甚至图片里的字也能直接识别翻译。这对每天要处理大量外文内容的人来说，省的时间是真实的。

---

## 三种核心翻译模式

TTime 最常用的三种方式，基本覆盖了日常 99% 的场景：

**划词翻译（Alt + E）**：鼠标选中文字，按下快捷键，翻译结果直接弹出来。看英文文章、处理邮件时用这个最顺手。

**截图翻译（Alt + W）**：截一下图，软件自动 OCR 识别里面的文字，然后翻译出来。PDF 里的文字、截图里的代码注释，统统搞定。

**输入翻译（Alt + Q）**：手动唤出翻译窗口，直接打字翻译。临时查个词或者翻一段自己写的内容用这个。

除了这三种，还有**替换翻译**（翻译结果直接替换选中内容）、**折叠翻译**（先不翻，点了再展开）、**悬浮球模式**（选中文字后冒出个小球，点了再翻）。根据自己的使用习惯，总有一款适合。

## AI 应用定制

会员版里有一个挺有意思的功能：**给 AI 设置角色预设和指令**。

比如你可以创建一个"代码解释"应用，遇到看不懂的代码，截图一下，AI 直接给你解释；或者"文字润色"应用，把自己写的中文翻译成英文后，再让 AI 给你优化一遍。不只是翻译工具，更像是一个定制化的 AI 助手。
